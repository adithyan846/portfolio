## 4. Aura Drone (Pixhawk + Raspberry Pi Integration)
**Description:** Integrated a **Pixhawk 2.4.8** flight controller with a **Raspberry Pi 5** to enable onboard AI processing.  

**Features:**
- Human/object detection using custom-trained **YOLOv11n** model.  
- Telemetry exchange via **MAVLink** between Pixhawk and Raspberry Pi.  
- Designed for disaster management use cases such as locating humans and coordinating delivery drones.  

**Key Skills:** Robotics, embedded Linux, ROS2 (Jazzy), computer vision, MAVLink communication, and AI integration.  
**Outcome:** Achieved real-time AI-driven decision support for drone autonomy.  
**GitHub Repository:** [github.com/adithyan846/aura.git](https://github.com/adithyan846/aura.git)
